# Wedding Surprise Cinematic V3

## Asset names
Place these files inside the `assets` folder:

- image1.png
- image2.png
- image3.png
- image4.png
- image5.png
- video1.mp4

Optional audio files:

- music.mp3
- chime.mp3

If you do not add audio files, the website still works.

## GitHub Pages
Upload all files to your GitHub repository root:

- index.html
- style.css
- script.js
- assets/
- README.md

Then enable GitHub Pages from Settings > Pages > Deploy from branch > main > /root.

## Flow
This V3 version does not behave like a slideshow. It waits for the user to click or type before moving forward. Typed responses are reused in emotional reveal messages.
