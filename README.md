# Wedding Surprise Cinematic MVP

## Files to upload into the assets folder

Put these exact files inside the `assets` folder:

- `image1.png`
- `image2.png`
- `image3.png`
- `image4.png`
- `image5.png`
- `video1.mp4`

The filenames must be lowercase exactly as written above.

## GitHub Pages hosting

1. Create a new public GitHub repository.
2. Upload all project files and folders into the repository root.
3. Go to Settings > Pages.
4. Select "Deploy from a branch".
5. Select `main` and `/root`.
6. Save.
7. Open the GitHub Pages link once deployment finishes.

## Notes

- Music and ambient effects require the user to tap "Begin the Surprise" because browsers block autoplay audio.
- The site is static and works on GitHub Pages.
- Keep the `index.html`, `styles.css`, and `script.js` files in the repository root.
- Keep all media files inside the `assets` folder.
